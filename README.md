> **Markets don't move on facts. They move on narratives. We built the machine that reads them.**

myboon is a mobile-first narrative intelligence app for on-chain traders and prediction market participants. A multi-agent brain watches prediction markets and on-chain signals 24/7, clusters them into narratives, stress-tests them, and surfaces only what matters — directly to your feed.

---

## How it works

```
Polymarket Collector  →  signals table  →  Analyst (every 15min)
                                         →  narratives table
                                         →  Publisher (every 30min)
                                         →  published_narratives
                                         →  Feed API (VPS)
                                         →  myboon app
```

No human touches it. A signal comes in, the Analyst picks it up, the Publisher decides if it's worth your attention, and it lands in your feed — automatically.

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm

### Install

```bash
git clone https://github.com/bucketshop69/myboon.git
cd myboon
pnpm install
```

### Run the API

```bash
cp packages/api/.env.example packages/api/.env
# fill in SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY
pnpm --filter @myboon/api start
```

### Run the mobile app

```bash
cd apps/hybrid-expo
pnpm start
```

Use a development build (not Expo Go) for wallet flows. See [`eas.json`](./apps/hybrid-expo/eas.json) for EAS build profiles including `dapp-store` (signed APK).

### Run the brain

```bash
cp packages/brain/.env.example packages/brain/.env
pnpm --filter @myboon/brain analyst   # runs once
pnpm --filter @myboon/brain publisher # runs once
```

## Infrastructure

- **Runtime:** Node.js / TypeScript (ESM)
- **Database:** Supabase (Postgres)
- **LLM:** MiniMax M2.5 (Anthropic-compatible API)
- **Mobile:** Expo (React Native)
- **Monorepo:** pnpm workspaces
- **VPS:** Collectors + API + Brain run on US VPS (Polymarket geo-restricted)
