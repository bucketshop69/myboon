Behavior notes - structure replaces nav

No top actions

The header only tells the user where they are. It does not ask them to switch, trade, connect, or manage anything.

No persistent bottom UI

The bottom of the screen belongs to the current card, CTA, sheet, or keyboard. There is no permanent bar fighting the scroll.

Sections are content

Feed, Markets, and Wallet are landings in the same home canvas. Their centered headings update the one real header while the user scrolls.

Wayfinding has scent

The first Feed chapter now includes a Today across myboon summary, so users learn Markets and Wallet exist without seeing nav chrome.

Route cards open apps

Each section preview has a destination-specific route card: Open full Feed, Browse all Markets, or Open Wallet.

Details still push

Tap a feed card, market, token, or position and it opens as a detail screen. Back returns to the same home landing.
